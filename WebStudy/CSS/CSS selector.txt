전체 선택자 : 문서 내 모든 요소를 선택합니다.
*{
  …스타일 정의…
}
문서 전체의 글꼴이나 글자 크기 등의 스타일 속성을 통일하고자 할 때 주로 사용
<body> 태그와 그 부모인 <html> 태그까지도 선택한다는 특징이 있어 문서의 초기 레이아웃을 설정할 때도 사용할 수 있습니다

태그 선택자 : 특정 태그로 만들어진 모든 요소를 선택합니다.
태그명{
  …스타일 정의…
}

클래스 선택자 : 특정 클래스 값이 지정된 모든 요소를 선택합니다.
/* 클래스 선택자 사용법 1 */
.클래스{
  …스타일 정의…
}
/* 클래스 선택자 사용법 2 */
태그명.클래스{
  …스타일 정의…
}
클래스가 여러 개 지정된 요소의 경우에는 클래스 선택자를 연달아 입력하는 방식으로 선택하는 것이 가능

아이디 선택자 : 특정 아이디 값이 지정된 요소를 선택합니다.
/* 아이디 선택자 사용법 1 */
#아이디{
  …스타일 정의…
}
/* 아이디 선택자 사용법 2 */
태그명#아이디{
  …스타일 정의…
}

그룹 선택자 : 여러 유형의 선택자를 한꺼번에 선택합니다
선택자1, 선택자2, …, 선택자n{
  …스타일 정의…
}

서로 다른 스타일 선언문의 선택자가 동일하게 입력된 경우, 웹 브라우저는 스타일 선언문 중 나중에 작성된 것을 요소에 적용
서로 다른 선택자가 동일한 요소를 가리키고 있는 경우, 앞서 확인했던 여러 선택자 유형 간의 우선 순위는 아이디 선택자, 클래스 선택자, 태그 선택자 순입니다.
