박스 모델은 너비와 높이를 가지는 개별 요소를 가리켜 ‘박스’라 표현하고, 박스를 네 가지 세부 영역으로 나누어 구분한 일종의 디자인 모델
CSS로 스타일을 정의할 수 있는 모든 요소에는 박스 모델 관련 속성 중 일부 또는 전부를 적용할 수 있으며, CSS는 영역 별로 크기를 따로 설정할 수 있도록 다양한 속성들을 제공
브라우저에서 문서를 열고 개발자 도구의 [Element] 메뉴에서 해당 요소를 선택한 다음, 개발자 도구 하단의 [Computed] 메뉴를 확인하면 요소의 박스 모델에 대한 이미지가 나타납니다.
콘텐츠 영역과 안쪽 여백, 그리고 경계선 까지가 요소의 일부로써 화면에 표시됩니다. 박스 모델의 경계선 바깥쪽에 있는 영역인 바깥쪽 여백은 요소의 형태나 콘텐츠를 정의하지 않고, 요소가 다른 요소와 얼마만큼 거리를 둘지 결정하는 역할

콘텐츠 영역 : 요소 내에 포함된 콘텐츠를 표시하는 영역으로, 관련 속성은 width과 height입니다.
px : 화면을 구성하는 단위인 화소(pixel)를 뜻하는 절대 단위
% : 부모 요소의 너비 또는 높이를 백분율로 계산해 적용하는 상대 단위
모든 웹 요소는 박스 모델에 기초해 콘텐츠 영역, 안쪽 여백, 경계선, 그리고 바깥쪽 여백을 가집니다. 그런데 인라인 요소의 경우에는 ‘콘텐츠 크기만큼만 공간을 차지한다’는 특성을 가지고 있기 때문에 
인라인 요소의 콘텐츠 영역에는 width나 height 속성이 따로 적용되지 않습니다.

경계선(테두리) : 콘텐츠와 안쪽 여백을 둘러싸는 경계선으로, 관련 속성은 border입니다.
화면 상에서 요소가 차지하고 있는 공간을 뚜렷하게 표시하고, 다른 요소와의 구분을 명확하게 할 수 있도록 하는 유용한 영역
+ border-width : 경계선의 두께를 지정합니다.
thin : 경계선을 얇게 표시합니다.
medium : 경계선을 중간 두께로 표시합니다.
thick : 경계선을 두껍게 표시합니다.
CSS 공식 명세서에서는 border-width의 각 키워드에 대한 정확한 두께를 정의하고 있지 않습니다. 각 키워드의 두께는 상대적이며 작업 환경에 따라 달라질 수 있음을 참고해주세요.
+ border-style : 경계선의 모양을 지정합니다. 점선, 실선, 겹선 등 다양한 모양을 선택할 수 있습니다.
none : 경계선을 표시하지 않습니다. 기본값입니다.
solid : 경계선을 실선으로 표시합니다.
dotted : 경계선을 점선으로 표시합니다.
dashed : 경계선을 짧은 직선으로 표시합니다.
double : 경계선을 이중으로 표시합니다. 두께가 커질수록 이중 경계선 사이의 간격이 커집니다.
groove : 경계선을 홈이 파인 듯한 입체적인 형태로 표시
+ border-color : 경계선의 색상을 지정합니다.
+ border : 단축 속성입니다. 경계선의 두께, 모양, 색상을 한꺼번에 지정할 수 있습니다.
+ 상하좌우 경계선 스타일 개별 지정하기
  <style>
    div{
      width: 100px;
      height: 100px;
      border-top: 1px solid red;
      border-right: 5px dotted orange;
      border-bottom: 10px dashed yellow;
      border-left: 20px double green;
    }
  </style>
+ 각 면의 서로 다른 속성값을 한 번에 지정하기
  <style>
    div{
      width: 100px;
      height: 100px;
      border-width: 1px 5px 10px 20px;
      border-style: solid dotted dashed double;
      border-color: red orange yellow green;
    }
  </style>
속성값 한 개일 때 : 모든 경계선에 동일한 값을 지정
속성값 두 개일 때 : 위아래 경계선과 왼쪽오른쪽 경계선의 값을 각각 따로 지정
속성값 세 개일 때 : 윗쪽 경계선, 왼쪽 오른쪽 경계선, 아래쪽 경계선 순서로 값을 지정
성값 네 개일 때 : 윗쪽, 오른쪽, 아래쪽, 왼쪽 순서대로 각각을 따로 지정
+ 모서리를 둥글게 만드는 border-radius
  <style>
    /* 
    절대 단위 사용
    좌상단 모서리부터 시계 방향으로 지정하기!
    */
    .abs{ border-radius: 10px 30px 50px 100px; }

    /* 상대 단위 사용 */
    .rel{ border-radius: 50%; }
  </style>

안쪽 여백 : 콘텐츠 주변을 감싸는 여백으로, 관련 속성은 padding입니다.
바깥쪽 여백 : 경계선 바깥을 감싸는 여백으로, 관련 속성은 margin입니다.
+ 상하좌우 여백 개별 지정하기
속성값 한 개일 때 : 여백의 모든 면에 동일한 값을 지정
속성값 두 개일 때 : 위아래 여백과 왼쪽오른쪽 여백의 값을 각각 따로 지정
속성값 세 개일 때 : 윗쪽 여백, 왼쪽 오른쪽 여백, 아래쪽 여백 순서로 값을 지정
속성값 네 개일 때 : 윗쪽, 오른쪽, 아래쪽, 왼쪽 순서대로 각각을 따로 지정

width와 height의 적용 방법을 결정하는 box-sizing
content-box : 기본값으로, width와 height가 콘텐츠 영역의 크기를 결정하도록 합니다. 이 경우 안쪽 여백이나 테두리 크기를 추가 적용하면 요소의 크기가 더 커지게 됩니다.
border-box : width와 height가 안쪽 여백과 테두리까지의 크기를 결정하도록 합니다. 이 경우 안쪽 여백이나 테두리 크기를 추가 적용하면 콘텐츠 영역의 크기가 좁아지게 됩니다.

박스 모델의 배경
박스 모델에 배경으로 추가된 색상이나 이미지 등은 박스 모델의 바깥쪽 여백을 제외한 나머지 영역 안에서 요소의 콘텐츠를 뒷받침합니다.
background-color : 요소의 배경 색상을 지정합니다.

background-clip : 배경 색상의 요소 내 적용 범위를 지정합니다.
+ 속성
border-box : 기본 값으로, 배경색이 박스 모델의 경계선까지 표시됩니다.
padding-box : 배경색이 박스 모델의 안쪽 여백까지 표시됩니다.
content-box : 배경색이 박스 모델의 콘텐츠 영역에만 표시됩니다.
text : 배경색이 텍스트 위에만 표시됩니다.

background-image : 요소의 배경 이미지를 지정합니다.
+ background-image: url(이미지의_URL);
 배경 이미지는 자신이 속한 박스의 크기에 맞춰 자동으로 크기를 변경하지 않고, 이미지 파일 본래의 크기를 유지한다는 특징이 있습니다. 이때 배경 이미지 지정 시 박스의 크기가 배경 이미지보다 큰 경우에는 배경 이미지가 여러 번 반복되어 표시되고, 
 박스의 크기가 배경 이미지보다 작은 경우에는 배경 이미지가 잘려 보인다는 특징 또한 존재합니다. 

background-origin : 배경 이미지의 요소 내 적용 범위를 지정합니다.
+ 속성
border-box : 배경 이미지가 경계선에서부터 채워지는 방식으로 표시됩니다.
padding-box : 기본 값으로, 배경 이미지가 안쪽 여백부터 채워지는 방식으로 표시됩니다.
content-box : 배경 이미지가 콘텐츠 영역부터 채워지는 방식으로 표시됩니다.

background-size : 배경 이미지의 크기(너비와 높이)를 지정합니다.
+ 속성
크기 지정 : 크기를 나타내는 수치와 단위를 함께 입력해 크기를 조절합니다. 너비와 높이에 해당하는 수치를 함께 입력할 수 있습니다. 
* cover : 이미지의 크기를 비율이 흐트러지지 않는 선에서 최대 크기로 조절합니다. 가로 또는 세로 방향에서 잘리는 영역이 발생할 수 있습니다. 
* contain : 이미지의 크기를 비율이 흐트러지지 않고, 잘리지도 않는 선에서 최대 크기로 조절합니다.

background-position : 배경 이미지의 위치를 지정합니다.
+ 속성
하는 위치값을 직접 수치와 단위로 나타내거나(좌측 상단 기준), 위치를 나타내는 키워드(top, bottom, right, left, center)를 지정할 수 있습니다.

background-repeat : 배경 이미지의 반복 여부 및 방식을 지정합니다.
+ 속성
repeat : 기본 값입니다. 배경 영역을 채울 때까지 이미지를 반복합니다. 마지막 반복 이미지가 영역을 벗어날 시 잘라냅니다.
repreat-x : 배경 영역의 너비를 모두 채울 때까지 이미지를 반복합니다. 반복은 가로 방향으로만 진행됩니다.
repreat-y : 배경 영역의 높이를 모두 채울 때까지 이미지를 반복합니다. 반복은 세로 방향으로만 진행됩니다.
no-repeat : 이미지가 반복되지 않고 한 번만 표시됩니다.
space : 이미지가 잘리지 않는 선에서 최대 개수를 반복합니다. 이미지는 양끝 정렬되고 이미지와 이미지 사이에 자동으로 여백이 추가됩니다.
